# daily-work-primer

Daily planning for AI solo operators.

This is a Codex Skill for people who work for themselves, work with AI, and need a clear morning ritual to set the day before the day eats them.

It is not a todo-list template.

It helps you:

- find the main contradiction
- choose today's main battlefield
- put important things first
- turn vague intent into observable outcomes
- create a not-doing list
- keep what worked yesterday
- remove what did not work
- iterate your personal work system every day

## Who It Is For

This skill is for AI solo operators, independent builders, one-person companies, creators, freelancers, researchers, and self-directed workers.

The shared condition is simple:

You are no longer only executing tasks someone else gives you. You are also designing the system that decides what should be done.

## Install

Copy the `daily-work-primer` folder into your Codex skills directory:

```text
~/.codex/skills/daily-work-primer
```

Then start a conversation like:

```text
Use daily-work-primer. Help me set today's work plan.
```

## Core Idea

Every morning, the skill helps you answer:

```text
What is today's most important battle?
What is the main contradiction?
What should be protected?
What should be ignored?
What did yesterday teach me?
```

The goal is not to finish everything.

The goal is to make the day point in the right direction.

## Output

The skill produces a daily plan with:

- today's tone
- main battlefield
- main contradiction
- key tasks
- not-doing list
- time rhythm
- daily iteration
- closing standard
